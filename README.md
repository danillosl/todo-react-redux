# todo-spring-boot

Simple todo using the flux architecture with react and redux and a few more things.

### Instalation

```sh
git clone https://github.com/danillosl/todo-react-redux.git
cd todo-react-redux
npm install
npm start
```
or
```sh
git clone https://github.com/danillosl/todo-react-redux.git
cd todo-react-redux
yarn install
yarn start
```
The project should be available at: http://localhost:8080 .
